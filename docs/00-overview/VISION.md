# Project Vision

Home Configurator is a reusable 3D spatial interface framework for Home Assistant.

It transforms smart-home control from a collection of cards into one persistent cinematic environment. Rooms become navigable contexts. Devices become interactive objects. Capabilities determine controls. Home Assistant remains the authoritative automation and state platform.

## Product goals

- Generate a useful dashboard automatically from Home Assistant areas, devices and entities.
- Allow every automatic decision to be edited without hardcoding.
- Support iPad landscape as the primary deployment target while remaining responsive.
- Package the product for HACS distribution.
- Support generic, manufacturer-specific and user-provided 3D models.
- Keep configuration portable through presets and mapping requirements.
- Preserve local-first operation and user privacy.

## Non-goals

- Copy proprietary source code or assets from reference sites.
- Replace Home Assistant automations or entity state.
- Become dependent on one apartment, manufacturer or entity naming scheme.
- Reproduce a conventional Lovelace card dashboard.
