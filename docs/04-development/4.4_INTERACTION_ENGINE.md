## Ownership flow

```text
Browser input
→ InteractionEngine
→ InputEngine normalization and arbitration
→ semantic target intent
→ selection/navigation/controller
→ Graphics Engine camera or scene update
```

## Gesture safety

One-finger orbit and two-finger continuous control are recognized as separate sessions. Pointer-count changes cancel or safely re-arbitrate the active gesture. The input engine suspends new claims during transition commits and releases all sessions on disposal.

## Accessibility

The 3D stage is focusable. Enter and Space activate the selected object, Escape cancels, and arrow keys expose deterministic rotation alternatives. Device controls added later must continue providing DOM equivalents for every gesture.
