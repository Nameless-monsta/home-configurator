# 4.6.1 — UI Foundation

**Status:** Complete and CI validated  
**Milestone version:** 0.6.1

## Delivered

- reusable `@home-configurator/ui` package;
- full-screen application shell with one persistent visual stage;
- safe-area-aware iPhone and iPad layout;
- compact, regular and wide layout classification;
- centralised overlay ownership;
- diagnostics overlay with keyboard dismissal;
- runtime status presentation;
- theme tokens and dark cinematic visual foundation;
- reduced-motion handling;
- focus-visible keyboard treatment;
- screen-reader labels for the stage and diagnostics;
- dashboard integration with Runtime, Graphics, Interaction and Home Assistant engines;
- unit tests for responsive layout classification.

## Scope boundary

This milestone established the shell and layout foundation. Room navigation, device navigation, configurator controls and device panels were delivered in later 4.6 milestones.

## Validation

GitHub Actions passed Prettier formatting, ESLint, strict TypeScript, unit tests and production builds.
