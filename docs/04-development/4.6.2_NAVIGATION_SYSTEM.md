# 4.6.2 — Navigation System

**Status:** Implemented, pending CI validation  
**Milestone version:** 0.6.2

## Delivered

- generic room and device navigation contracts;
- deterministic navigation state model;
- automatic first-room and first-device selection;
- room and device selectors with accessible listbox semantics;
- back and forward history;
- keyboard traversal with Arrow Up, Arrow Down, Home, End and Escape;
- focus-visible treatment;
- compact, regular and wide responsive presentation;
- unavailable-device labelling;
- live Home Assistant room and device mapping in the dashboard;
- safe fallback when selected rooms or devices disappear;
- unit tests for selection, history and fallback behaviour.

## Scope boundary

This milestone does not include configurator controls, device panels or final motion choreography. Those remain in 4.6.3 onward.

## Validation gate

The milestone is complete only after GitHub Actions passes formatting, ESLint, strict TypeScript, tests and production builds.
