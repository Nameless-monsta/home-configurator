# 4.6.6 — iyO Motion System

Status: implementation complete; CI validation pending.

## Delivered

- Shared motion tokens for duration, easing and travel distance.
- Framework-independent `UiMotionController` built on the Web Animations API.
- Automatic cancellation of superseded animations.
- Panel entrance and exit choreography.
- Section and newly rendered content reveals.
- Button press micro-interactions.
- Toggle and segmented-control feedback.
- Mutation-driven orchestration for overlays, navigation and configurator content.
- Operating-system reduced-motion support.
- CSS fallback and compositing hints.
- Dashboard lifecycle integration.
- Unit tests for tokens, choreography, cancellation and reduced motion.

## Architecture

The motion controller is independent from Home Assistant, Three.js and any rendering framework. `UiMotionOrchestrator` observes the stable DOM contracts exposed by the UI package and delegates all animation to the controller.
