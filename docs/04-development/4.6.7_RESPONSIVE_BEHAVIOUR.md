# 4.6.7 Responsive Behaviour

Status: implementation complete, validation pending.

## Delivered

- Responsive profile model using width, height, orientation and pointer precision.
- Phone, tablet and desktop viewport classes.
- Portrait, landscape and short-height handling.
- Root data attributes for layout, viewport, orientation, height and pointer type.
- Phone bottom-sheet configurator and navigation menus.
- Tablet portrait configurator behaviour.
- Desktop side-panel behaviour.
- Safe-area padding for notches and home indicators.
- Dynamic viewport-height units for mobile browser chrome.
- Minimum 44 px targets for coarse-pointer devices.
- Narrow-device overflow protection.
- Responsive unit tests.

## Out of scope

- Final accessibility and visual regression audit.
- Final UI integration validation across the full browser/device matrix.

Those are handled by 4.6.8.
