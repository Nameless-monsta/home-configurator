## Interaction contract

### Colour

- Horizontal movement changes hue with circular wrapping across 0–360 degrees.
- Vertical movement changes saturation and clamps it to 0–100 percent.
- Intermediate updates are marked as continuous.
- Pointer release emits the final command.

### Brightness

- Upward two-finger movement increases brightness.
- Downward two-finger movement decreases brightness.
- Mouse-wheel input provides the desktop equivalent.
- Brightness remains clamped to the canonical 0–1 range.

## State flow

```text
Gesture intent
    ↓
Gesture command adapter
    ↓
Semantic Home Assistant command
    ↓
Optimistic Runtime layer
    ↓
Selected-device model binding
    ↓
Authoritative confirmation or rollback
```

The interaction target does not mutate the Three.js model directly. The model continues to render from Runtime `effectiveState`, preserving one authoritative application state path.
